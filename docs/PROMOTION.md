# 掲載営業・プロモーション手引き

「タイピングゲーム」の検索上位は個別ゲームではなく**まとめ記事**が占めている。
三番手への現実的な道は「まとめ記事・ポータルに載る → 被リンクと流入を得る →
指名検索『ゾンビタイピング』を増やす」の順。掲載依頼は個人開発ゲームでは
**ごく一般的な活動**で、失礼にはあたらない(メディア側もネタを探している)。

## 送る前のチェックリスト

- [ ] 独自ドメインを確定させた(URLが後で変わると掲載URLが古くなる。docs/SEO.md 参照)
- [ ] Search Console でインデックス済み
- [ ] OGP画像でシェア時の見栄えを確認済み(𝕏に自分でURLを貼ってカード表示を確認)

## 掲載先の候補(上から優先)

### 1. 無料ゲーム紹介ポータル(登録型・審査あり)

- **ふりーむ!** … ブラウザゲーム登録可。個人開発の定番
- **フリーゲーム夢現** … 同上。レビュー文化があり被リンクも得られる
- **Plicy(プリシー)** … ブラウザゲーム投稿プラットフォーム
- **itch.io** … 海外向けだが「HTML5 game」として登録可能。プロフィールから被リンク
- **unityroom は Unity 専用のため対象外**

### 2. タイピング関連のまとめ記事・ブログ

検索「タイピングゲーム おすすめ」「タイピングゲーム 無料 ブラウザ」で上位に出る
記事の運営元に問い合わせフォーム経由で連絡する。記事末尾に運営者情報があることが多い。

### 3. 𝕏(旧Twitter)・配信者

- ハッシュタグ: #タイピングゲーム #ブラウザゲーム #個人開発 #indiegame
- 30秒のプレイ動画(ハードコアの絶望感 or エンドレスの物量)を添付すると反応が段違い
- タイピング系・作業配信系のVTuber/配信者に「配信で遊んでOK、収益化もOK」と明記して送る

## 注意

- 一斉BCCではなく1通ずつ、相手の記事名に触れて送る(コピペ感があると載らない)
- 返信がなくても2週間空けて1回だけリマインド。それ以上は追わない
- 掲載されたら𝕏で感謝を添えて拡散(メディア側の実績にもなり、次が載りやすくなる)
